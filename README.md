Uso de IA:
Principalmente se utilizó Github Copilot para mejorar el diseño base, agregando sombras y adaptando cada sección agregada al diseño estético elegido.
Los prompts en un inicio se hicieron en inglés para adaptarse mejor al uso de IA que solo acepte ingreso de prompt en este mismo idioma, después se optó por seguir en español neutro, para un mejor entendimiento de las ordenes por parte propia.
Además se utilizó este programa para mejorar el posicionamiento de las imágenes dentro de la página, y mejorar la eficiencia en torno al volumen de texto en cada sección, quitando o agregando espacio o secciones, solo en caso de ser necesario.
